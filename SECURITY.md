# Security Policy

## Purpose

This repository is an educational Bash scripting resource.

## Reporting Issues

If you find an error, outdated information, broken command, or security concern, please open an issue or submit a pull request.

## Responsible Use

Only run scripts on systems you own, administer, or have explicit permission to use.

Do not use scripts from this repository for unauthorized access, credential capture, persistence, evasion, log tampering, or offensive actions.

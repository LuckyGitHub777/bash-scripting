# Archived Unsafe Scripts

This folder documents why certain older script ideas should not be maintained as runnable examples in this beginner repository.

The following patterns are not appropriate for this repo:

- Keylogging or shell-history exfiltration
- Reverse shell payloads
- Log clearing or anti-forensics

These items are treated as unsafe for a beginner Bash foundations repository.

They should not be executed, improved, expanded, or presented as operational tooling.

If studying defensive detection or incident response, use authorized training materials, controlled labs, and legal/ethical guidelines.
